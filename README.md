# Leonard O. Lalican - Portfolio

A modern, professional portfolio website built with React.js

## 🚀 Quick Start

### Prerequisites
- Node.js (Download from https://nodejs.org/)

### Installation
```bash
cd portfolio-react
npm install
```

### Run Locally
```bash
npm start
```
Opens at http://localhost:3000

### Deploy to GitHub Pages
```bash
npm run deploy
```

See `portfolio-react/DEPLOYMENT.md` for detailed deployment instructions.
